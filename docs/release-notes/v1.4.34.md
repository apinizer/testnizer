## v1.4.34

**Switching sidebar pages no longer disturbs your open tabs, tool inputs stick
around — and Test Suite runs now report what your tests actually say.**

- **Global tab strip:** open tabs and the active tab are now shared across every
  sidebar page. Clicking **Tools** (or Tests / Mocks / History) no longer hides
  your request tabs, deselects the active one, or loses unsaved edits — the page
  only swaps the left panel, and you come back to exactly the tab you were on.
- **Tool inputs persist:** JWT / Diff / Hash and the other tools keep what you
  typed when you switch to a request tab and back; they used to reset because the
  tool only stayed mounted while it was the active tab.
- **Text Diff wrapping:** long lines in the diff result now wrap within their
  column instead of overflowing and overlapping the neighbouring text.
- **Test Suite verdict:** a request whose test explicitly accepts a non-2xx code
  — e.g. an idempotent DELETE asserting the status is one of 200 / 204 / 404 /
  400 — now counts as **passed** in the run summary, matching what the Tests tab
  shows (and Postman / Insomnia). The summary used to bucket every 4xx/5xx as
  failed even when the test passed.
- **Insomnia scripts read the response again (issue #47):** imported
  after-response scripts that check `insomnia.response.status` / `.body` /
  `.json()` now work during a collection Run. The importer used to rewrite
  `insomnia.*` to `pm.*`, which silently turned the numeric status into the
  reason-phrase text — so a guard like `if (status < 200 || status >= 300)`
  threw before the body was read, and body assertions failed while the status
  check still passed.
- **Multipart file uploads attach again (issue #46):** a `form-data` file field
  bound to a path (e.g. an MMDB upload) is now sent with the file and a correct
  multipart boundary. An unreadable path surfaces a clear error naming the field
  and path instead of being silently dropped into a confusing server 400.

**Tests:** sidebar-page-switch coverage; runner-verdict parity (idempotent
DELETE passes, a bare 4xx with no test still fails); Insomnia `.status` stays
numeric so body assertions run; multipart file-from-path attaches and errors
loudly on a missing path.
