## v1.4.34

**Switching sidebar pages no longer disturbs your open tabs, and tool inputs
stick around.**

- **Global tab strip:** open tabs and the active tab are now shared across every
  sidebar page. Clicking **Tools** (or Tests / Mocks / History) no longer hides
  your request tabs, deselects the active one, or loses unsaved edits — the page
  only swaps the left panel, and you come back to exactly the tab you were on.
- **Tool inputs persist:** JWT / Diff / Hash and the other tools keep what you
  typed when you switch to a request tab and back; they used to reset because the
  tool only stayed mounted while it was the active tab.
- **Text Diff wrapping:** long lines in the diff result now wrap within their
  column instead of overflowing and overlapping the neighbouring text.

**Tests:** sidebar-page-switch coverage — the active tab and the full tab set are
unchanged across every page switch, and unsaved request edits survive a Tools
round-trip.
